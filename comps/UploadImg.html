# Attributes 属性

参数 说明 类型 可选值 默认值
placeholder 占位 string - '请选择'
disabled 是否将该分组下所有选项置为禁用 boolean - false
IDS 文件fileId Array - []
accept 接受上传的文件类型 string - 'image/*'
btnIsShow 用作图片预览时隐藏按钮 boolean - true
widthHeight 需要显示的宽高 [string, number] - 100
limit 最大上传数量 number - 1
maxSize 最大文件大小 number - 5
imgList 回显文件, 必传 Array - []
submitSuccess 是否上传成功 boolean - false
