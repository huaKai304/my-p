# Methods

方法名称 说明 参数
addressToLnglat 地址信息转经纬度 接收2个参数，1.地址信息 2. 回调函数，返回1个参数包含经纬度的对象
lnglatToAddress 经纬度转地址信息 接收2个参数，1.经纬度数组 2. 回调函数，返回1个参数地址信息
getCurrentPosition 获取当前位置经纬度(成功前提，有定位硬件且有定位授权) 接收1个参数，1. 回调函数，返回1个包含经纬度的对象
