# Attributes 属性

参数 说明 类型 可选值 默认值
isScroll 是否滚动 boolean - false
fontSize 字体大小(px) number - 14
placeholder 提示 string - '请选择'
appendToBody 是否挂载到body boolean - false
ID 单选时传入 [String, Number] - -
IDS 多选时传入 Array - []
NAME 单选时传入 string - -
NAMES 多选时传入 Array - []
disabled 是否禁用 boolean - false
multiple 是否多选 boolean - false
flat 是否可选任意节点 boolean - true
apiName 请求接口名称 string - 'getAllDeptTree'
apiParams 请求参数，当接口所需参数改变时，重新获取树结构 Object - {}
childrenName 子级数组的名称 string - 'childrenDepartment'
labelName 子级label string - 'departmentValue'
disableBranchNodes 是否阻止选择分支节点 boolean - false
isDisabled 用于禁用项目选择 boolean - true
valueConsistsOf 在多选模式下，数组中应包括哪种节点 具体参考文档 'Branch priority' string - 'BRANCH_PRIORITY'
