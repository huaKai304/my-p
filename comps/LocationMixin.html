# LocationMixin

# 基本用法

使用(互转地址信息和经纬度)前准备 (opens new window)

<script>
import { LocationMixin } from "szzt-ui"
export default {
    mixins: [LocationMixin],
    mounted() {
        // 地址信息转经纬度
        this.addressToLnglat("北京市海淀区苏州街", (lnglat) => {
            console.log(lnglat)
        })
        // 经纬度转地址信息
        this.lnglatToAddress([116.305583, 39.981536], (address) => {
            console.log(address)
        })
        // 获取当前位置经纬度
        this.getCurrentPosition((lnglat) => {
            console.log(lnglat)
        })
    }
}
</script>

# Methods

方法名称 说明 参数
addressToLnglat 地址信息转经纬度 接收2个参数，1.地址信息 2. 回调函数，返回1个参数包含经纬度的对象
lnglatToAddress 经纬度转地址信息 接收2个参数，1.经纬度数组 2. 回调函数，返回1个参数地址信息
getCurrentPosition 获取当前位置经纬度(成功前提，有定位硬件且有定位授权) 接收1个参数，1. 回调函数，返回1个包含经纬度的对象
