# Eslint

以下只列出部分，具体详细看官网

# eslint (opens new window)

  1. space-before-blocks 在块之前强制保持一致的间距

  2. space-in-parens 在括号内强制保持一致的间距

  3. space-infix-ops 运算符周围需要间距

  4. no-debugger 不允许debugger语句

  5. no-unused-vars 禁止未使用的变量

  6. no-var 需要 let 或 const 而不是 var

  7. indent 强制一致的缩进，默认4 spaces

  8. eqeqeq 判断是否相等只能使用===和!==

# eslint-plugin-vue (opens new window)

  1. vue/max-attributes-per-line template模板元素单行及多行的属性个数

  2. vue/attribute-hyphenation template模板元素属性名以kebab-case书写

  3. vue/component-definition-name-casing 强制name属性PascalCase格式书写

  4. vue/html-closing-bracket-newline 强制在标记的右括号之前使用换行符（或不使用换行符）。

  5. vue/html-end-tags 禁止缺少结束标签

  6. vue/html-self-closing 没有内容的标签可以省略关闭标签

  7. vue/no-multi-spaces 删除标记中不用于缩进的多个空格。

  8. vue/no-shared-component-data data属性为函数

  9. vue/attributes-order 强制对组件属性进行排序。默认顺序在 Vue.js 风格指南中指定
